| Command                  | Description                                                             |
| ------------------------ | ----------------------------------------------------------------------- |
| **BASICS**         |                                                                         |
| REM `<text>`           | Comment.`<text>` is any string. This line will not be executed.       |
| DELAY `<milliseconds>` | Waits for `<milliseconds>` (integer) before running the next command. |
| DEFAULT_DELAY            | Sets default delay between all keystrokes.                              |
| STRING `<text>`        | Types `<text>` as keyboard input.                                     |
|                          |                                                                         |
| **PRESSING KEYS**  |                                                                         |
| ENTER                    | Presses the Enter key.                                                  |
| TAB                      | Presses the Tab key.                                                    |
| ESC                      | Presses the Escape key.                                                 |
| BACKSPACE                | Presses the Backspace key.                                              |
| SPACE                    | Presses the Spacebar                                                    |
| UP                       | Presses the Up Arrow key.                                               |
| DOWN                     | Presses the Down Arrow key.                                             |
| LEFT                     | Presses the Left Arrow key.                                             |
| RIGHT                    | Presses the Right Arrow key.                                            |
| CAPSLOCK                 | Toggles the Caps Lock key.                                              |
| DELETE                   | Presses the Delete key.                                                 |
| F1-F12                   | Presses function key F1 through F12.                                    |
| CTRL `<key>`           | Holds Control and presses `<key>`.                                    |
| SHIFT `<key>`          | Holds Shift and presses `<key>`.                                      |
| ALT `<key>`            | Holds Alt and presses `<key>`.                                        |
| WINDOWS `<key>`        | Holds Windows/Command key and presses `<key>`.                        |
|                          |                                                                         |
|                          |                                                                         |
| REPEAT `<count>`       | Repeats the previous command `<count>` times                          |
